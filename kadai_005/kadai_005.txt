##Step1. ローカルリポジトリ「git_challenge」を作成する

PS C:\samurai\git_terakoya> mkdir git_challenge


    ディレクトリ: C:\samurai\git_terakoya


Mode                 LastWriteTime         Length Name
----                 -------------         ------ ----
d-----        2025/08/12     20:06                git_challenge


PS C:\samurai\git_terakoya> cd git_challenge
PS C:\samurai\git_terakoya\git_challenge> git init
Initialized empty Git repository in C:/samurai/git_terakoya/git_challenge/.git/


##Step2. テキストファイル「hello_world.txt」を作成する

PS C:\samurai\git_terakoya\git_challenge> git add hello_world.txt
PS C:\samurai\git_terakoya\git_challenge> git commit -m ".\hello_world.txtを追加"
[master 349f21f] .\hello_world.txtを追加
 1 file changed, 1 deletion(-)


##Step3. 「hello_world.txt」に2回の変更をコミットする

PS C:\samurai\git_terakoya\git_challenge> git add hello_world.txt  
PS C:\samurai\git_terakoya\git_challenge> git commit -m "Hello,worldを追記"
[master fa5017a] Hello,worldを追記
 1 file changed, 1 insertion(+)
PS C:\samurai\git_terakoya\git_challenge> git add hello_world.txt
PS C:\samurai\git_terakoya\git_challenge> git commit -m "こんにちは、世界を追記"
[master 37f9343] こんにちは、世界を追記
 1 file changed, 1 insertion(+), 1 deletion(-)


##Step4. 「hello_world.txt」の変更履歴を表示する

PS C:\samurai\git_terakoya\git_challenge> git log --oneline
